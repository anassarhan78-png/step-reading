# 🌍 STEP Reading

An interactive platform for improving English reading skills and preparing for the STEP exam.

---

## 🎯 About the Project

**STEP Reading** is designed to help learners develop strong reading comprehension skills through structured practice.

The platform provides:

* 📖 Carefully designed reading passages for multiple levels (A1, A2, B1, STEP)
* 🧠 STEP-style multiple-choice questions
* 🌐 Real-world topics inspired by academic and reliable sources

---

## 📊 Content Structure

| Level | Passages | Description                                                |
| ----- | -------- | ---------------------------------------------------------- |
| A1    | 20       | Beginner – simple vocabulary and basic sentence structures |
| A2    | 20       | Elementary – short texts on familiar topics                |
| B1    | 20       | Intermediate – longer texts with varied vocabulary         |
| STEP  | 25       | Advanced – academic texts similar to STEP exam             |

---

## 🧠 Learning Approach

Each passage includes:

* 5 structured paragraphs
* 5 comprehension questions:

  * Vocabulary
  * Main Idea
  * Reference
  * Detail (x2)

---

## 🚀 Getting Started

1. Open the website
2. Choose your level
3. Read the passage carefully
4. Answer the questions
5. Check your understanding
